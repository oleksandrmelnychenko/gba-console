> **STATUS: MIGRATION COMPLETE (2026-05-29).** The full Clients area was migrated to `gba_console` at functional parity on branch `clients/full-migration` across slices 0–10 (foundation, edit card all tabs, new-client wizard, pricing, structure, perfect-client, sales/recommendations, e-commerce, cash-flow drill-downs, list filters, supplier passport, tails). A final 10-screen completeness re-audit returned **0 residual gaps**. Each slice was verified with `tsc` + `eslint` + `build` + tests.
>
> Accepted (documented) deferrals are cross-module only: the full sale editor and Supply Logistics page (un-migrated `sales`/warehouse modules), OutcomePaymentOrder multi-task cash-flow enrichment + per-service supplier sub-template layouts (data shown generically), cross-reload wizard session persistence (replaced by route-safe state by design), and Poland-specific / legacy-disabled surfaces. The "what is NOT migrated" inventory below is retained as the historical pre-migration baseline.

Generated by a 45-agent fan-out (inventory → console-code audit → adversarial re-verification) over 15 client sub-areas.
